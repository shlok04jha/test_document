Ports Description
=================

Inputs
------



.. list-table::
   :header-rows: 1
 

   * - Signal
     - Width
     - Description
   * - operator_i
     - ALU_OP_WIDTH
     - Operation selector generated by decoder
   * - operand_a_i
     - 32 bits
     - First operand
   * - operand_b_i
     - 32 bits
     - Second operand

Outputs
-------



.. list-table::
   :header-rows: 1
   

   * - Signal
     - Width
     - Description
   * - adder_result_o
     - 32 bits
     - Result of internal adder
   * - result_o
     - 32 bits
     - Final ALU output
   * - comparison_result_o
     - 1 bit
     - Comparator output
   * - is_equal_result_o
     - 1 bit
     - Equality flag
